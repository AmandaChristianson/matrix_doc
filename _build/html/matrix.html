matrix module#

Matrix object with operations.

The matrix is an object which stores the elements of the matrix in a nested list. The library comes with both methods as part of the class and external functions useful for matrices.

The methods use Python’s standard notation for instance constructing, addition, subtraction and multiplication. The standard multiplication is implemented to support both matrix- and scalar-matrix multiplication, depending on input. Additional functionality include element-wise multiplication and division as well as computing the trace and transpose of a matrix. Finally, the function identity(n) constructs an identity matrix with dimension n*n.

__mul__(x)[source]#

Matrix or matrix-scalar multiplication. Called by writing “self * x” where self is a matrix. Returns the result of scalar-matrix multiplication if x is a scalar (float or int). Returns the result of matrix multiplication self * x if x is a matrix.

Requirement for matrix multiplication: self._columns == x._rows.

Parameters: self = matrix, x = matrix, int or float Returns: nested list

Time complexity of scalar multiplication: O(m*n) where m*n is the dimension of the matrix. Time complexity of matrix multiplication: O(m*n*k) where m is the number of rows in self, n number of columns in self (=number of rows in x) and k number of columns in x.

__rmul__(x)[source]#

Scalar-matrix multiplication if x is a scalar (float or int). Called by writing “x * self” where self is a matrix, and x is not. Parameters: self=matrix, k=int or float Returns: nested list Time complexity: O(m*n) where m*n is the dimension of the matrix.
